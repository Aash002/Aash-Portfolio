# Aashish Waghela — Premium Portfolio

This is a static, premium portfolio website ready for Vercel, Netlify, GitHub Pages or any free static host.

All required downloadable files are kept in this single folder. There are no nested asset folders.

## Files

- `index.html` — website
- `styles.css` — visual design and responsive layout
- `tokens.css` — design tokens
- `script.js` — reveal interactions
- `AashishWaghela_CV.pdf` — downloadable CV

## Run locally

Open `index.html` in your browser, or run:

```bash
python3 -m http.server 5173
```

Then visit:

```text
http://localhost:5173
```

## Deploy on Vercel

1. Create a GitHub repository.
2. Upload all files from this folder directly into the repo root.
3. In Vercel, click **Add New → Project**.
4. Import the repository.
5. Select **Other** as framework preset.
6. Leave build command empty.
7. Output directory: `.`
8. Deploy.
